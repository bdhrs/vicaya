## What surprised me

This strongly cuts against the pure cessationist reading. Worth leading with in any future note on this topic.

2. **Paranum gaps in Udāna, Itivuttaka, MN49.** Several passages (including the famous *viññāṇaṃ anidassanaṃ* verse) returned empty `paranum` from the canon search. The CST tables for these books apparently don't carry consistent paragraph numbering. This means `resolve-citation` can't verify them — citations had to be sutta-name-only, which is less precise.

Either the search terms didn't match the commentary's register, or the book isn't indexed as expected. This is a real gap — the Atthasālinī is the primary commentarial source for the Abhidhamma realist position.

## What I'd change in the skill

1. **Add a fallback when paranum is empty.** When `resolve-citation` can't return a human citation (paranum is empty or missing), the skill should instruct the agent to still include the passage in Canon Evidence but note the citation as "sutta name only, paranum unavailable." Currently this is ad-hoc.

2. **Atthasālinī search terms need explicit guidance.** The commentary uses different register than the sutta language. Future runs should try both `nibbāna` and `asaṅkhata` in commentary books (`*_att`), and explicitly check abh01a_att for Abhidhamma topics.

3. **Auto-caption handling note.** The "paraphrase, don't quote Pāḷi" rule is clear, but the note could explain _why_ (auto-caption transcription errors) rather than just silently paraphrasing. A parenthetical "(auto-caption transcript)" in citations would make the evidence quality transparent.
